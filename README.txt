Prueba Técnica - Banco de Bogotá
Descripción
Este repositorio contiene una aplicación web desarrollada con Angular para el frontend y Spring Boot para el backend. El propósito de este proyecto es demostrar habilidades técnicas en el desarrollo de aplicaciones web.

Requisitos de Ejecución
Para ejecutar la aplicación, asegúrate de tener instalados los siguientes requisitos en tu sistema:

Angular CLI: 11.2.19
Node.js: 10.18.0
Sistema Operativo: Windows (win32 x64)
Java: 1.8


Proceso de Ejecución

-Ejecutar el Backend

Navega al directorio del backend
Inicia la aplicación Spring Boot con el siguiente comando: mvn spring-boot:run


-Ejecutar el Frontend

Navega al directorio del frontend
Inicia la aplicación Angular con el siguiente comando:
ng serve


-Acceso a la Aplicación
Frontend: Accede a la aplicación en tu navegador en la siguiente URL:

http://localhost:4200/login

Backend: La API del backend estará disponible en:

http://localhost:8080
